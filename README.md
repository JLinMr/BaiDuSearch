# BaiDuSearch
i0.wp.com是wp的静态图片CDN加速永久缓存速度很快呀建议使用 

使用方法  i0.wp.com/你的网站/你的图片  

ps：只能加载图片

例子
https://i0.wp.com/sy.bsgun.cn/img/favicon.ico
